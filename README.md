# 📘 ip-2026-01

Este repositório reúne todas as listas de exercícios e questões desenvolvidas durante a disciplina de **Introdução à Programação**, cursada no semestre **2026/01**.

---

## 🧠 Sobre a disciplina

A disciplina tem como objetivo principal desenvolver o raciocínio lógico e a base de programação, incluindo:

- Lógica de programação
- Estruturas de decisão e repetição
- Vetores e matrizes
- Funções
- Manipulação de algoritmos matemáticos
- Introdução à resolução de problemas computacionais

---

## 💻 Linguagens utilizadas

### 🟡 Portugol (inicialmente)
No início da disciplina, os conceitos foram trabalhados utilizando **Portugol**, com foco em:

- Estrutura de algoritmos
- Lógica básica
- Compreensão de fluxo de execução

### 🔵 Go (linguagem principal do semestre)
Após a base inicial, os exercícios passaram a ser implementados na linguagem **Go (Golang)**, utilizada para:

- Implementação de algoritmos
- Resolução de listas de exercícios
- Aplicação prática dos conceitos de lógica

## 🏢 Projeto Integrador

O **Projeto Integrador** é uma iniciativa que integra os conhecimentos adquiridos nas disciplinas do primeiro semestre do curso de Engenharia de Software da UFG. Ele visa aproximar os alunos das situações reais de desenvolvimento de software, incluindo não só a parte computacional, mas também a metodologia do processo de desenvolvimento.

A empresa parceira apresenta um **problema real**, e os alunos devem:

1. **Analisar o problema proposto** pela empresa.
2. **Aplicar os conhecimentos adquiridos** nas disciplinas de **Introdução à Programação** e outras componentes do curso.
3. **Desenvolver uma solução computacional** utilizando a linguagem **Go**, com foco em **lógica, organização** e **pensamento computacional aplicado**.
4. **Propor uma implementação funcional** que atenda às necessidades e requisitos da empresa.
5. Trabalhar com **metodologias de desenvolvimento** de software, como ciclos iterativos, testes, controle de versões, entre outros.

O objetivo é **aproximar o aprendizado acadêmico de situações reais** de desenvolvimento de software, estimulando a **resolução de problemas práticos**, além de proporcionar uma experiência completa de desenvolvimento de soluções, desde a análise até a implementação.
